# wwwtxt

> Unix fortune, but with early internet posts

## Install
```
npm install -g wwwtxt
```

## Usage
```
wwwtxt
```

Will print a random tweet.

```
wwwtxt --cow
```

Will print a random tweet using [cowsay](https://www.npmjs.com/package/cowsay)
